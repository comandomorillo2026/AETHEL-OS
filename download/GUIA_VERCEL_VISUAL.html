Guía de Deployment en Vercel

¿Por qué Vercel?

✨ 100% Gratis
🚀 Creado para Next.js
🔒 SSL Incluido
⚡ Deployment Automático
🌍 CDN Global
1

Crear Cuenta en GitHub

GitHub es donde se guarda tu código. Vercel lo lee desde ahí.

  1. Ve a github.com
  2. Click en "Sign Up"
  3. Completa tu registro (es gratis)
Ir a GitHub →
2

Subir tu Proyecto a GitHub

Opción más fácil (Manual):

  1. En GitHub, click en "New Repository"
  2. Nombre: nexusos
  3. Click "Create repository"
  4. Click "uploading an existing file"
  5. Arrastra todos los archivos del proyecto
  6. Click "Commit changes"
💡 Tip: Descarga el ZIP del proyecto y extrae los archivos antes de subirlos.
3

Crear Cuenta en Vercel

  1. Ve a vercel.com
  2. Click en "Sign Up"
  3. Elige "Continue with GitHub"
  4. Autoriza a Vercel para acceder a tu GitHub
Ir a Vercel →
4

Hacer Deployment

  1. En Vercel, click "Add New Project"
  2. Selecciona tu repositorio nexusos
  3. Click "Import"
  4. Vercel detectará automáticamente Next.js
  5. Click "Deploy"
⏱️ Espera 2-3 minutos mientras se compila tu proyecto...
5

Configurar Base de Datos

Necesitas una base de datos PostgreSQL. Opciones gratuitas:

Supabase

supabase.com

500MB gratis

Neon

neon.tech

3GB gratis

Vercel Postgres

Integrado

256MB gratis

⚠️ Después de crear la base de datos, copia la DATABASE_URL y agrégala en:
Vercel → Tu Proyecto → Settings → Environment Variables
