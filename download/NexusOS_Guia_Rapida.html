🚀 Pasos para Correr el Servidor

1. Abrir terminal en la carpeta del proyecto:

cd /home/z/my-project

2. Instalar dependencias (solo primera vez):

npm install

3. Correr el servidor:

npm run dev

4. Abrir en el navegador:

http://localhost:3000
